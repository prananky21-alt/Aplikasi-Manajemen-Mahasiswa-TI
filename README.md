# Aplikasi Manajemen Data Mahasiswa TI

Aplikasi web sederhana untuk mengelola data mahasiswa menggunakan Python + Streamlit.

## Fitur Utama
1. **CRUD Mahasiswa**: Tambah, Edit, Hapus, Tampilkan data
2. **Login Multi-Role**: Admin & User dengan password hashing SHA256
3. **File I/O**: Simpan data ke `data_mahasiswa.json` dan `users.json`
4. **OOP**: Class `Person`, `Mahasiswa`, `Auth`, `ManajemenMahasiswa`
5. **Algoritma**: Linear Search, Bubble Sort, Selection Sort
6. **Validasi**: Regex untuk NIM, Nama, Username
7. **Error Handling**: Try-Except untuk file & input
8. **Backup Data**: Download JSON langsung dari aplikasi
